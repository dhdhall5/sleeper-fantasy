# sleeper-fantasy

Dynasty fantasy football dashboard powered by Sleeper API, FantasyCalc values, and AI trade analysis.

## Features
- 14-team SuperFlex dynasty league dashboard
- My Team tab with AI chat + analysis
- League Home, All Teams, Trade Analyzer, Waiver Wire tabs
- FantasyCalc dynasty SuperFlex player values
- AI-powered trade recommendations (Option A/B/C tiers)
